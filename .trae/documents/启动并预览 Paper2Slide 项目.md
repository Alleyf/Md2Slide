## 启动并预览项目计划

该项目是一个基于 Vite 和 Remotion 的 React 项目。我将按照以下步骤启动并预览项目：

### 1. 确认并安装依赖
虽然项目中已存在 `node_modules`，但我将首先运行 `npm install` 以确保所有依赖项都已正确安装并处于最新状态。

### 2. 启动 Vite 开发服务器
运行 `npm run dev` 启动交互式演示。
- **命令**: `npm run dev`
- **默认端口**: `3000` (根据 `vite.config.ts` 配置)
- **用途**: 预览交互式论文教学演示系统。

### 3. 预览项目
服务器启动后，我将使用预览工具打开 `http://localhost:3000` 供您查看。

### 4. (可选) 启动 Remotion 预览
如果您需要查看具体的视频动画效果，我还可以为您启动 Remotion 预览。
- **命令**: `npm run remotion:preview`
- **用途**: 专门用于查看和调试 Remotion 视频组件的动画。
