# 自定义快捷键功能测试

## 测试目标
验证编辑器中的自定义快捷键功能是否正常工作，包括：

1. 复制当前行 (Ctrl+D)
2. 删除当前行 (Ctrl+Shift+D) 
3. 向上移动行 (Alt+Up)
4. 向下移动行 (Alt+Down)
5. 格式延续功能 (Enter)
6. 自定义快捷键设置功能

## 测试步骤

### 1. 复制当前行测试
- 在编辑器中输入几行文本
- 将光标放在某一行
- 按下 Ctrl+D
- 验证该行是否被复制到下一行

### 2. 删除当前行测试
- 在编辑器中输入几行文本
- 将光标放在某一行
- 按下 Ctrl+Shift+D
- 验证该行是否被删除

### 3. 移动行测试
- 在编辑器中输入多行文本
- 将光标放在中间某一行
- 按下 Alt+Up 或 Alt+Down
- 验证该行是否上下移动

### 4. 格式延续测试
- 在编辑器中输入 `- 列表项`
- 按下 Enter 键
- 验证下一行是否自动延续列表格式

### 5. 自定义快捷键测试
- 打开设置菜单
- 进入键盘快捷键设置
- 修改某个快捷键绑定
- 验证修改后的快捷键是否生效

## 预期结果
所有上述功能都应该按照预期工作，自定义快捷键功能允许用户修改默认的快捷键绑定。

## 实现细节
此功能通过以下文件实现：
- `src/services/keyboardService.ts` - 快捷键管理服务
- `src/components/KeyboardShortcutsPanel.tsx` - 快捷键设置界面
- `src/App.tsx` - 编辑器键盘事件处理
- `src/types/keyboard.ts` - 快捷键类型定义
