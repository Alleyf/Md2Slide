# Markdown2Slide - 3Blue1Brown 风格幻灯片生成器

一个基于 React 和 Remotion 风格设计的 Markdown 转幻灯片工具。它专注于提供 3Blue1Brown 风格的视觉体验，支持数学公式、代码高亮和动态交互。

## ✨ 特性

- 🎨 **3Blue1Brown 视觉风格**：深色背景、明亮配色与优雅的排版。
- ⚡ **实时预览**：左侧编辑 Markdown，右侧实时更新幻灯片。
- 📊 **数学公式支持**：完美集成 LaTeX，支持块级和行内公式。
- � **代码语法高亮**：内置多种编程语言的代码展示支持。
- 🎮 **动态交互**：支持内容逐步显示（Click-to-reveal）和键盘导航。
- � **一键导入**：支持导入本地 `.md` 文件。

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

启动后，访问 `http://localhost:3000` 即可开始创作。

## 📁 Markdown 语法说明

### 基础元素
- `# 标题`：幻灯片主标题
- `## 副标题`：幻灯片副标题
- `---`：**幻灯片分隔符**（用于切换下一页）
- `- 列表项`：自动支持逐步显示动画

### 高级功能
- `$$公式$$`：块级 LaTeX 公式
- `$公式$`：行内 LaTeX 公式
- `!icon(emoji)`：插入大尺寸装饰图标
- `!grid`：显示坐标网格（3B1B 风格）
- `!vector`：显示示例向量动画（3B1B 风格）
- `!video(url)`：插入视频内容

## 🎮 导航快捷键

- `空格` 或 `→`：下一步 / 下一页
- `←`：上一步 / 上一页

## 🔧 技术栈

- **React**：UI 框架
- **Vite**：构建工具
- **React-Markdown**：Markdown 解析
- **KaTeX**：数学公式渲染
- **Emotion**：样式管理
